**On the Trail** is an AI-powered detective mystery game where every case is completely unique. Built with React (Vite) and powered by OpenAI, the game dynamically generates characters, stories, and clues - creating a fresh mystery every time you play.

Players step into the role of a detective investigating unusual cases, questioning suspects, gathering clues, and trying to uncover the truth before the suspects lose their patience.

Live Demo:
👉 https://on-the-trail.vercel.app/

---

## ✨ Features

- 🧩 Procedurally generated mysteries — every game creates a new story, location, and suspects
- 👥 Unique AI characters — interact with suspects with different personalities and traits
- 💬 AI-powered conversations — natural and dynamic dialogue powered by OpenAI
- 😰 Stress system — suspects may stop talking if pushed too hard
- 🔎 Detective gameplay — gather clues, analyze statements, and identify the culprit
- 🌍 Multilingual support using i18next

## 📦 Installation

Follow the steps below to run the project locally.

### 1. Clone the Repository

```bash
git clone https://github.com/PawelRomik/on-the-trail.git
cd on-the-trail
```
### 2. Setup Environment Variables
Create a `.env` file in the root directory and add:
```bash
OPENAI_API_KEY=your_openai_api_key
```
### 3. Install Dependencies
```bash
npm install
```
Then install Vercel CLI:
```bash
npm install -g vercel
```
### 4. Run the Project
Start the development environment with:
```bash
vercel dev
```
The application will start locally and you can open it in your browser.
